## Tools Grabber KTP Multi User
 Studensite Universitas Kuningan


## Instalasi
```
git clone https://github.com/aceptriana/tools/
cd tools
```
```
pip install selenium
```
```
pip install requests
```
* isi user.txt & pass.txt terlebih dahulu 
```
python ktp.py
```
